Keepcon.com

Web Design | Responsive | HTML/CSS+PHP

Keepcon wanted to update their website. I redesigned it completely from scratch, gave it a modern look and added a PHP contact form.


The Project

Keepcon is a highly-advanced plaform for automated customer experience management. However their website –unlike their technology– wasn’t very modern.

The Goals

  • Modernize the looks of the website
  • Update its content to reflect newly developed products
  • Make it responsive
  • Create a way for interested visitors to get in touch

Mobile First Approach

Digging into Google Analytics, I found an important percentage of visitors were using mobile browsers. The new design had to be completely responsive without compromising the experience nor the content.




Complexity


The platform provides lots of services and tools for handling big data. To give a more immersive experience, I used large imanigery that showcased the way data was displayed.

I had to create all the images and mockups from scratch. To keep loading times acceptable, I ran all the code and images through optimizers and minifiers using Gulp.js.

PHP Form


I tried several plugins for the contact form but none of them blended with the design the way I wanted. So I decided to create the form myself.

For a few days, I commited to learn about Mailchilmp’s API, Mandrill and basic PHP so we could use the current mail delivery system to send the data from the contact form to the marketing department’s inbox. To make better use of the space, the desktop version displays both views at the same time.

The Result

Give the scope of the project and the amount variety of tools I had to use, this is the the work I am most proud of and the one in which I’ve learned the most. You can see the website live at www.keepcon.com.
